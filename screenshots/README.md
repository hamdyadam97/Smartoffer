# دليل Screenshots

## كيفية التقاط الصور
1. افتح النظام في المتصفح (Chrome/Edge).
2. اذهب للصفحة المطلوبة.
3. اضغط `F12` أو `Ctrl+Shift+I` وفعل **Device Toolbar** لو عايز تقص شاشة موبايل.
4. اضغط `Ctrl+Shift+S` (أو استخدم أداة Snipping Tool في Windows) للتقاط الصورة.
5. احفظ الصورة في هذا المجلد بنفس الاسم المذكور.
